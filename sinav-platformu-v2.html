İlk Kurulum
Sınavların tüm cihazlardan erişilebilir olması için ücretsiz Supabase
veritabanı kurmanız gerekiyor. Aşağıdaki adımları takip edin.
1Supabase hesabı aç
supabase.com adresine gidin → "Start your project" → GitHub ile
ücretsiz kayıt olun.
2Yeni proje oluştur
"New project" tıklayın, bir isim ve şifre girin, bölge olarak eu-
central-1 seçin. Proje oluşturması ~2 dakika sürer.
EduQuiz
3Tabloları oluştur
Sol menüden SQL Editor'a gidin, aşağıdaki kodu yapıştırıp
çalıştırın:
create table quizzes (
id uuid default gen_random_uuid() primary key,
class_id text not null,
branch text not null,
name text not null,
time_limit integer default 0,
questions jsonb not null default '[]',
created_at timestamptz default now()
);
alter table quizzes enable row level security;
create policy "Public read" on quizzes
for select using (true);
create policy "Anon insert" on quizzes
for insert with check (true);
create policy "Anon update" on quizzes
for update using (true);
create policy "Anon delete" on quizzes
for delete using (true);
4API bilgilerini girin
Sol menüden Project Settings → API'ye gidin. Project URL ve
anon public key'i aşağıya yapıştırın.
