Ilk Kurulum

Sinavlarin tum cihazlardan erisilebilir olmasi icin ucretsiz Supabase veritabani kurmaniz gerekiyor. Asagidaki adimlari takip edin.

1

Supabase hesabi ac

supabase.com adresine gidin → "Start your project" → GitHub ile ucretsiz kayit olun.

2

Yeni proje olustur

"New project" tiklayin, bir isim ve sifre girin, bolge olarak eu-central-1 secin. Proje olusturmasi ~2 dakika surer.

3

Tablolari olustur

Sol menuden SQL Editor'a gidin, asagidaki kodu yapistirip calistirin:

create table quizzes ( id uuid default gen_random_uuid() primary key, class_id text not null, branch text not null, name text not null, time_limit integer default 0, questions jsonb not null default '[]', created_at timestamptz default now() ); alter table quizzes enable row level security; create policy "Public read" on quizzes for select using (true); create policy "Anon insert" on quizzes for insert with check (true); create policy "Anon update" on quizzes for update using (true); create policy "Anon delete" on quizzes for delete using (true);
4

API bilgilerini girin

Sol menuden Project Settings → API'ye gidin. Project URL ve anon public key'i asagiya yapistirin.
